Buzzbold Coding Exam Requirements

This exercise will be used to understand and evaluate how you complete a small task using HTML, JavaScript and a JSON-based API. Buzzbold would like to see your progress at various points throughout the exercise; please check your code into your repository at least four times throughout the process of the exercise. For each commit, comment in such a manner that any other developer could step in and take over the codebase and continue in this process.

Instructions:

  1. Fork this repository to a new GitHub repository.
  2. Create a form in index.html that requires the following inputs (each with a maximum length of 30 characters per input):
    • Email Address (the format of the email address must be valid)
    • First Name (alphabetic string greater than 1 character in length)
    • Last Name (alphabetic string greater than 1 character in length)
    • Date of Birth (using ISO 8601 standard)
  3. The form should have a submit button that is only enabled when all of the above fields have been completed to specification. If a user enters an invalid value in a form input, an appropriate error message should be displayed when the focus of the input is blurred.
  4. When a user submits the form, an AJAX POST request will be sent to http://dev.buzz.do/code_exam/. The value of the responseString from the response to this request should be displayed in a dynamically created paragraph element that becomes the last element inside of the form element.
  5. Within the code, the form should always be treated as an object.
  6. If there is a failure of any sort, a message should be displayed asking the user to try again.

The format of the AJAX request should be:

{ "data" : 
 [{
    "email" : "[theemail]",
    "first_name" : "[thefirstname]",
    "last_name" : "[thelastname]",
    "birth_date" : "[thebirthdate]" 
  }]
}
